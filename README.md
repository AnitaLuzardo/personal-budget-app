# Personal Budget App

## Crear Base de Datos
Se utilizó **MySQL** versión 8 y el backup se encuentra en el siguiente directorio:
> database/backup/

Los datos del usuario se encuentran en el siguiente archivo
> database/config/config.js
## Instalar

```
$ npm i
```
## Correr la aplicación

```
$ node src/app
```
> http://localhost:3001
